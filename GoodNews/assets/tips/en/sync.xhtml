]> Synchronization

GoodNews knows two types of syncs:

push sync:
Pushes local changes (e.g. tag assignments, read state changes) to google reader. If the immediate push is enabled in the sync settings this will be done automatically whenever there are pending changes. Can be triggered manually through the menu.
full sync:
A push sync followed by a pull to ensure that your local articles, tags and subscriptions are in sync with google reader. Can be manually triggered using the sync button from the action bar or automatically by activating the continous sync in the settings or by using Tasker, Locale or compatible apps.
